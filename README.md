# Predictive-Marketing-Dashboard
Predictive Marketing Dashboard using python and tableau
This project presents an end-to-end Predictive Marketing Dashboard built using Tableau and Python.
It combines data visualization, business KPIs, and machine learning to analyze customer behavior and optimize marketing campaign strategies.
The dashboard enables users to:
Track key marketing KPIs
Analyze customer spendig patterns
Identify high-value customer segments
Predict customer response to campaigns

Objectives
-Analyze customer purchasing behavior
-Measure marketing effectiveness using KPIs
-Apply machine learning to predict campaign response
-Build an interactive dashboard for data- driven decision making

Tools and technologies
Tableu desktop- data visualization and dashboard creation
Ptyhon- data preprocessing and machine learning
Pandas/ NumPy- data manipulation
Scikit learn- predictive modeling

Data set
The data set includes customer - level marketing data such as:
-Income
-Total spending
-Number of purchases
-Campaign Interactions
-Customer demographics

Data processing (Python)
-Handled missing values
-Created new features:
  -Total sprending
  -Total purchases
  Engagement metrics
-Built a predictive model to classify customer response

Machine learning
A classification model was developed to predict whether a customer will respond to a marketing campaign.
Output:
-Prediction (0/1)
-Used in tableau for visualization and segmentation

Dashboard features
-Key performance indicators (KPIs)
 -Total Customer Spending
 -Conversion Rate
 -Response rate

-Customer behavior analysis
  - Scatter plot: Income vs spending
  - Trend line showing correlation
  - Color segmentation based on predicted response

-Predictive insights
  -Bar chart showing
    -Customers likely to response
    -Customer unlikely to responde

Key Insights
-Customers with higher income tend to spend more
-A small segment of customers shows high probability of campaign response
-Marketing efforts can be optimized by targeting high- value segments
